# 🏆 台股多週期產業板塊資金流向與黑馬雷達地圖 (Sector Performance & Money Flow Map)

這是一個專為台股交易者設計的**頂級視覺化產業板塊資金流向監控地圖與黑馬雷達看板**。它可以自動獲取台股 1970+ 檔個股的最新收盤與量能數據，並結合價量特徵，即時追蹤市場資金輪動、資金淨流入/流出，以及主力默默低調吸籌的「緩漲黑馬」板塊。

---

## 🌟 核心特色功能

### 1. 📊 頂級 ECharts 互動式產業板塊樹狀圖 (Treemap)
* **市值權重與多週期漲跌**：方塊面積代表個股或板塊之市值規模，顏色代表漲跌幅度（翠綠代表下跌，亮紅代表上漲，符合台股標準）。
* **無縫多週期切換**：支援一鍵切換 **今日漲跌 (1D)**、**週累積 (5D)**、**雙週累積 (10D)** 及 **月累積 (20D)**，提供不同維度的短中線趨勢視角。
* **縮放與層級下鑽 (Drill-down)**：支援滑鼠滾輪與觸控板捏合手勢放大縮小，點擊中型族群可平滑放大並展示次產業與個股明細，點擊葉節點個股可於側邊欄 focus 定位。

### 2. ⚡ 智慧批次分析連動 (Bulk Analysis Filter)
* 支援直接貼上 TradingView 或看盤軟體的選股清單（自動過濾單字母、百分比、價格等欄位雜訊）。
* **Treemap 與清單動態連動**：貼上清單分析後，整個資金地圖與側邊欄排行會動態收縮，**僅顯示名單內個股的產業結構與漲跌幅**，並重新計算專屬的板塊均值。

### 3. 🔥 資金流向與黑馬雷達 (Money Flow Radar)
結合價量結構與統計學公式，後台自動計算各板塊的資金面指標：
* **量能比 (VER, Volume Expansion Ratio)**：衡量板塊當前週期平均成交額相較於 20 日基準均量的放大倍數（例如 `1.5x` 代表量能擴增至均量的 1.5 倍）。
* **5日資金比重走勢 (Sparkline 📊)**：在每個中型族群標題旁，直接嵌入一個極度精緻的 **5日資金比重微型走勢圖 (Sparkline)**。柱子越高代表比重越大，柱子漸增代表資金正持續流入。
* **🐢 默默緩漲指標 (Quiet Risers)**：透過 **趨勢持續性指標 (TCI)**（收紅天數佔比 $\ge 60\%$）與日均波動度門檻，自動標註並挑選出股價溫和墊高、波動度極低、主力暗中低調吸籌的「緩漲黑馬」族群，並在板塊旁亮起綠色 **「🐢 緩漲黑馬」** 勳章。
* **側邊欄雷達看板**：
  * **🔥 資金淨流入**：找出期間內量增價揚、放量追價最顯著的板塊。
  * **⚠️ 資金淨流出**：找出放量下跌、主力撤退的危險板塊。
  * **🐢 默默緩漲**：找出中長線緩步墊高的默默建倉黑馬（於 5D / 10D / 20D 模式下啟用）。

### 4. 🗂️ 精緻校正之三層級高保真（High-fidelity）產業分類體系
* 將全台股 1970+ 檔股票拆解為 20 個大產業、近 100 個精細的次產業。
* 經過嚴格的人工校正與大掃除，合理解決了「單一股票自成次產業」的破碎問題，對 ABF 載板、CoWoS 設備、CPO 光通訊、重電與電線電纜、半導體特用化學品等前瞻科技板塊做出了精準的收攏。

## ⚙️ 快速開始

### 1. 安裝依賴套件

本專案只需安裝三個核心 Python 套件：
```bash
pip install -r requirements.txt
```

### 2. 執行更新並編譯看板

在專案資料夾下執行：
```bash
python track_daily_performance.py
```
執行後，程式會自動：
1. 從 yfinance 下載台股最新行情（下載 45 日歷史以確保 20D 中長線指標與滾動均線的計算品質）。
2. 計算 1D、5D、10D、20D 資金比重、量能比 (VER)、收紅比 (TCI) 及默默緩漲 (QRS) 分數。
3. 生成 `daily_sector_performance.html`（網頁看板）與 `daily_sector_performance.md`（文字報告）。

### 3. 💡 Windows 懶人一鍵啟動
您只需在 Windows 檔案總管中**按兩下執行 `run_tracker.bat`**，它會自動完成數據更新，並立即在您的預設瀏覽器中打開最新的資金地圖網頁！

## 📂 核心檔案結構說明

* `track_daily_performance.py`: 看板生成的主控程序（包含網頁範本、量價指標運算與資料下載）。
* `classify_all_database.py`: 核心大數據分類定義與特徵比對引擎。
* `industry_cache.json`: 台股 1970+ 檔股票的基本面與產業簡介快取，避免重複下載。
* `run_tracker.bat`: 一鍵更新並開啟網頁看板的批次檔。
* `daily_sector_performance.html`: 編譯出來的動態資金地圖網頁（可直接用瀏覽器打開）。
* `app.py`: Streamlit 雲端網頁進入點。

## 💡 指標與圖例速查
* 🐢 **緩漲黑馬**：過去 5D/10D/20D 累積溫和上漲、收紅天數佔比高（&ge; 60%）、日均波動度低，通常為主力默默進貨之低調起漲族群。
* 📊 **5日資金比重走勢 (Sparkline)**：顯示過去 5 日該板塊佔大盤成交金額的比重走勢。柱子越長代表比重越高；柱子持續走高代表資金正在持續流入該板塊。
* 🔥 **資金淨流入**：板塊上漲且量能比 (VER) 大於 1.0 的板塊，分數越高代表量價俱揚越顯著。
* ⚠️ **資金淨流出**：板塊下跌且量能比 (VER) 大於 1.0 的板塊，分數越低代表放量下殺越嚴重。
